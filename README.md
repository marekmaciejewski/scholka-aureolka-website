# Scholka Aureolka Website

React, TypeScript, and Vite project for the Scholka Aureolka website.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` type-checks and builds the production bundle.
- `npm run lint` runs ESLint.
- `npm run preview` serves the production build locally.

## Project Structure

- `src/main.tsx` mounts the React app.
- `src/App.tsx` is the top-level application shell.
- `src/index.css` contains global baseline styles.
- `public/Logo*.svg` contains the prepared logo assets.

The generated Vite demo content has been removed so feature work can start from a clean shell.
